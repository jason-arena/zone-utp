# ZonaUTP Marketplace

¡Perfecto! El marketplace está listo con diseño minimalista y todas las funcionalidades frontend básicas.

## ✅ Lo que se ha creado:

### 📱 **Marketplace Principal** (`marketplace.html`)
- **Header completo** con logo, búsqueda, navegación y menú de usuario
- **Hero section** con estadísticas de la plataforma
- **Sistema de filtros** por categorías, precio, ubicación y ordenamiento
- **Grid de productos** responsivo con vista de cuadrícula y lista
- **Tarjetas de productos** con toda la información necesaria
- **Footer** completo con enlaces y información

### 🎨 **Diseño y CSS**
- **`marketplace.css`** - Estilos principales con sistema de variables CSS
- **`components.css`** - Componentes reutilizables (modals, buttons, forms, etc.)
- **Paleta de colores UTP** (#6B46C1, #553C9A, #8B5CF6, #ffffffff)
- **Completamente responsivo** - Mobile, tablet y desktop
- **Diseño minimalista** y profesional
- **Sistema de CSS Grid** y Flexbox moderno

### ⚡ **JavaScript Funcional** (`marketplace.js`)
- **Búsqueda en tiempo real** con debouncing
- **Filtros dinámicos** por categoría, precio, ubicación
- **Sistema de favoritos** con localStorage
- **Modals interactivos** para contacto y venta
- **Toast notifications** para feedback
- **Vista grid/lista** intercambiable
- **Scroll suave** y efectos de animación
- **Carga dinámica** de productos (Load More)
- **Tooltips** y componentes UI avanzados
