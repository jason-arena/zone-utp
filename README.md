# ZonaUTP

ZonaUTP es un marketplace digital pensado exclusivamente para la comunidad universitaria de la UTP. Su objetivo es facilitar la compra, venta e intercambio de productos y servicios entre estudiantes, profesores y personal administrativo, en un entorno seguro, organizado y moderno.

## Características principales

- Acceso mediante correo institucional para mayor seguridad.
- Publicación de productos y servicios por categorías (libros, electrónicos, muebles, servicios, etc.).
- Búsqueda y filtros avanzados para encontrar rápidamente lo que necesitas.
- Chat privado entre usuarios para negociar y coordinar transacciones.
- Sistema de reputación y reportes para fomentar la confianza.
- Interfaz moderna, responsiva y fácil de usar, inspirada en Facebook Marketplace.
- Panel de usuario para gestionar publicaciones, historial y perfil.

## Objetivo

Modernizar la experiencia de compra y venta dentro de la UTP, brindando una plataforma centralizada, segura y eficiente para toda la comunidad universitaria.

## Tecnologías sugeridas

Frontend: HTML5, CSS3, JavaScript
Backend: PHP 8+ con patrón MVC
Base de datos: MySQL 8+
Entorno: XAMPP para desarrollo local
Producción: Preparado para Render

## ¿A quién va dirigido?

- Estudiantes
- Profesores
- Personal administrativo de la UTP
